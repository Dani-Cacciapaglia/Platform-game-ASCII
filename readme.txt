README

Il programma è pronto all'uso e non ha bisogno di installazioni ulteriori per il funzionamento.
All'interno della cartella si trovano i file .cpp e i file .hpp nonchè il file principale per l'esecuzione.
Il progetto è stato sviluppato sulla piattaforma "Code Blocks" e con l'ausilio delle librerie specificamente 
forniteci per l'occasione in aggiunta a quelle impiegate a lezione:
-conio.h
-windows.h

Per l'esecuzione del gioco fare doppio click sul file "ooo" e compilare sul proprio IDE (scritto e testato su
code blocks consigliamo per una riuscita più sicura l'impiego dello stesso IDE).


tasti disponibili = w, s, a, d, spazio
